脱敏类型说明

将整个字段值替换为相同长度的星号。

示例:

  • 13812345678 → ***********
  • test@example.com → *************
  • 张三 → **

参数: 无需参数

保留前 N 个字符和后 M 个字符，中间替换为星号。

示例:

  • 手机号：{"keep_first":3,"keep_last":4} → 13812345678 → 138****5678
  • 身份证：{"keep_first":6,"keep_last":4} → 110101199001011234 → 110101********1234
  • 邮箱：{"keep_first":2,"keep_last":0} → test@example.com → te*******
  • 姓名：{"keep_first":1,"keep_last":1} → 张三丰 → 张**丰

参数: {"keep_first":3,"keep_last":4}

说明:

  • keep_first: 保留前缀字符数
  • keep_last: 保留后缀字符数

使用正则表达式匹配字段值，匹配到的内容替换为星号或自定义格式。

示例:

  • 手机号脱敏： {"pattern": "(\\d{3})\\d{4}(\\d{4})","replace": "$1****$2"}
    原始：13812345678 → 结果：138****5678
  • 邮箱脱敏： {"pattern": "(\\w{2})\\w*(@.*)","replace": "$1****$2"}
    原始：rqwd@qq.com → 结果：rq****@qq.com
  • 身份证脱敏： {"pattern": "(\\d{6})\\d*(\\d{4})","replace": "$1********$2"}
    原始：110101199001011234 → 结果：110101********1234
  • 银行卡号脱敏： {"pattern": "(\\d{4})\\d*(\\d{4})","replace": "$1**********$2"}
    原始：6222021200012345678 → 结果：6222**********5678

参数: {"pattern": "\\d{3}(\\d{4})\\d{4}","replace": "$1"}

说明:

  • pattern: 正则表达式
  • replace: 替换格式（使用 $1 等引用分组）

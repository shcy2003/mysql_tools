- [ ] 没有顶部导航栏
- [ ] 没有左边侧边栏
- [ ] 登录卡片居中显示
- [ ] 有完整的渐变背景
- [ ] 可以正常登录

- [ ] 可以创建新的数据库连接
- [ ] 测试连接按钮可以正常工作

- [ ] 侧边栏可以展开/折叠数据库连接

- [ ] 点击表名可以在编辑器中生成SELECT语句（覆盖，不是追加）

- [ ] 显示查询执行时间
- [ ] 最多返回10条记录

- [ ] 查询结果后显示"导出Excel"按钮

- [ ] Excel文件包含正确的数据

## 查询历史
- [ ] 可以查看查询历史

- [ ] 可以创建脱敏规则
- [ ] 可以编辑脱敏规则
